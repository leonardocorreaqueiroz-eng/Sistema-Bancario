# Sistema Bancário

Projeto desenvolvido em Java 21 utilizando:

- Maven
- Hibernate
- JPA
- MySQL

## Funcionalidades

- Cadastro de clientes
- Conta corrente
- Conta investimento
- Depósitos
- Saques
- Transferências
- Rendimentos automáticos

## Configuração

Copie:

hibernate.cfg.example.xml

para:

hibernate.cfg.xml

e configure suas credenciais do banco.
